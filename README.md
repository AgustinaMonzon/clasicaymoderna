# 💈 Clásica & Moderna - Sistema de Turnos Online

Sitio web oficial de **Clásica & Moderna**, barbería de tradición en Victoria, Entre Ríos. Desarrollado para agilizar la reserva de turnos y fortalecer la presencia digital del negocio.

## 🚀 Funcionalidades Principales

- **Gestión de Turnos:** Formulario dinámico con validación de horarios y fechas en tiempo real.
- **Base de Datos:** Integración con Google Sheets a través de **SheetDB** para el manejo de turnos y almacenamiento de reseñas.
- **Sistema de Reseñas:** Los clientes pueden calificar el servicio con estrellas y dejar comentarios que se visualizan dinámicamente.
- **Optimización Mobile-First:** Interfaz diseñada para una experiencia fluida desde smartphones.
- **Geolocalización:** Acceso directo a la ubicación del local mediante Google Maps.

## 🎨 Diseño y Estética
El sitio utiliza una paleta de colores **Dark Mode** (negros y grises profundos) con acentos en **Dorado Cobrizo (#a67c52)**, evocando la elegancia de las barberías clásicas. 

Se destaca el **Sello de Confianza** en el pie de página, resaltando la trayectoria y el oficio de Elvio Monzón como firma de autor.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 y JavaScript (ES6+).
* **Fuentes:** *Playfair Display* (títulos) y *Roboto* (lectura clara).
* **Backend:** SheetDB API (Google Sheets como base de datos).
* **Deploy:** **Vercel** (Hosting y Continuous Deployment).
* **Control de versiones:** Git y GitHub.
